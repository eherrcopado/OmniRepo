Sales Analytics 5.0: Now with point-and-click customization, global filters for faster performance, and more

We’ve revamped Sales Analytics so you can add, edit, and delete KPIs through the point-and-click interface. Additional enhancements--including global filters and dynamic time period selectors--make the app faster and more precise than ever.

Sales Analytics helps your entire sales team unlock the full power of Sales Cloud data. With the app’s interactive dashboards based on best practice templates, KPIs, and pipeline trending, it’s never been easier to build a winning sales team. 

New With this Release 

-Point and Click Editing. Sales Analytics has been reworked to allow for easier editing of dashboards. You can add, edit, and delete KPIs without ever having to touch any code.

-Global Filters. To optimize performance and make dashboards even easier to customize, the dashboards listed below now use global filters instead of list widget filters. Global filters let you filter different KPIs in the same dashboard--for example, the performance of two sales reps--with one filter. You can add, edit, or remove filters by editing the global filter panel. And dashboards load more quickly because global filters require fewer queries than the previous list widgets.

-Precise time-period selection for the pipeline waterfall chart. Dynamically choose the start and end period for pipeline comparison using a relative date selector widget.  

-Updated content for the following dashboards:

-Sales Performance. We’ve updated average deal size, win rate, and other KPIs. We’ve also added an opportunity values table to make it easier to perform Salesforce actions directly from the dashboard.

-Stage Analysis. A new historical time frame filter lets you choose the time frame used for the historical time in stage calculation.

-Leaderboard. A new created date filter lets you filter on both the created and the close date of opportunities. KPI comparisons versus same time last year have been changed to comparison of selected sales rep to team average.

-Sales Rep Overview. Comparison KPIs have been removed with the addition of global filters, but the dashboard is fully editable using the point-and-click Analytics interface.
